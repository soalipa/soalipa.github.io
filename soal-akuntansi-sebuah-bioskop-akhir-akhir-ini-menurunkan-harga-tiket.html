Model matematika yang sesuai dengan masalah ini dapat ditulis sebagai persamaan linear berikut:

20,000x + 25,000y = 3,250,000

Di mana x adalah banyak penonton dengan harga tiket Rp20.000,00 dan y adalah banyak penonton dengan harga tiket Rp25.000,00. Dengan menggunakan persamaan ini, kita dapat mencari nilai x dan y dengan memasukkan batasan seperti bahwa jumlah penonton tidak boleh kurang dari 0 dan tidak boleh lebih dari 150.

12. Sebulan yang lalu harga tiket ke kereta api ke sebuah kota Rp300.000,00 dan jumlah penumpang sebanyak 400 orang. Sementara itu pada akhir pekan yang akan datang ketika bertepatan dengan libur panjang, dengan harga tiket yang sama, ternyata jumlah penumpang melonjak menjadi 800 orang bahkan ada yang tidak kebagian tiket. Dari data tersebut, jenis elastisitas yang terjadi pada harga tiket adalah....


Koefisien Elastisitas Permintaan

∆Q / ∆P x P / Q

________________________________

Diketahui :

P1 = Rp300.000

Q1 = 400 orang

P2 = Rp300.000

Q2 = 800 orang

∆Q = Q2 - Q1

∆Q = 800 - 400

∆Q = 400

∆P = P2 - P1

∆P = 300.000 - 300.000

∆P = 0

Ditanya :

Ed = ?

Jawab :

Ed = ∆Q / ∆P x P / Q

Ed = 400 / 0 x 10.000 / 500

Ed = ∞ x 20

Ed = ∞ (elastis sempurna)

Jadi, dari data tersebut, jenis elastisitas yang terjadi pada harga tiket adalah elastis sempurna

31. dalam sebuah gedung bioskop terdapat 150 tempat duduk. harga tiket di kursi bagian depan Rp 20.000 dan harga tiket di kursi bagian belakang Rp 15.000. jika pada hari itu bioskop penuh dan pendapatanya Rp 2.500.000, berapa banyak penonton bagian depan


Banyak penonton di bagian depan adalah 50 orang.

Pembahasan

Diketahui :

Banyak tempat duduk = 150 tempat duduk

Harga tiket tempat duduk bagian depan = Rp. 20.000

Harga tiket tempat duduk bagian belakang = Rp 15.000

Pendapatan = Rp 2.500.000

Ditanya : Banyak penonton di bagian depan

Dijawab:

Tempat duduk bagian depan = x

Tempat duduk bagian belakang = y

x + y = 150   ..... 1)

20.000 x + 15.000 y = 2.500.000

20 x + 15 y = 2.500  ..... 2)

Eliminasi persamaan 1 dan 2

x + y = 150               Ix20I  20x + 20y = 3000

20 x + 15 y = 2.500  Ix1I    20x + 15y = 2500 -

                                                       5y = 500

                                                          y = 100

Substitusikan nilai y ke dalam persamaan 1

x + y = 150

x + 100 = 150

x = 50

Jadi, penonton di bagian depan berjumlah 50 orang dan penonton dibagian belakang berjumlah 100 orang.
