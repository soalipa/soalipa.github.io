Ngartekake aksara jawa

Pacitan kutha Sewu siji guwa

Pacitan iku sawijining wewengkon kang kepencil adoh saka punjering kutha.

Lemah garapane rata rata cengkar, amerga tlatahe luwih saka telung Prapat kadadeyan saka pegunungan kapur.

Saka kahanan alam kang kaya mangkono iku mau Pacitan isi potensi sumber daya alam lan pariwisata.

Manut panaliten para ahli geologi ing Pacitan akah ditemokake tambang timah, tembaga, emas, uranium lan batubara.

Obyek pariwisata sing diandhalane yaiku wisata guwa lan pesisir.

__________________Pembahasan

Ing wacana dhuwur, nggunakake aksara murda lan aksara swara.

➢ Aksara murda, yaiku aksara sesirah kang sinebut huruf kapital. Aksara murda digawe kanggo pakurmatan utawa menghormati lan lumrahe kanggo nulis jenenge wong utawa pangkat lan drajat sarta papan Panggonan.

→ Aksara murda cacahe ana 8 :

Na = ꦟKa = ꦑTa = ꦡSa = ꦯPa = ꦦNya = ꦘGa = ꦓBa = ꦨ

➢ Aksara swara, yaiku aksara jawa kang digunakake kanggo nulis huruf vokal.

→ Aksara swara cacahe ana 5 :

a = ꦄi = ꦇu = ꦈe = ꦌo = ꦎ

★ catetan :

huruf sing kacethak kandel lan diwenehi garis ngisor iku sing nggunakake aksara murda lan aksara swara.

13. Tolong Artikan Dan Jawabkan Soal Bahasa Jawa Halaman 46 Progam Remedial 1 Sampe 5 Aksara Jawa Soalnya Besok Di Kumpulkan


1. bapak tindak kanto ( kantor )

2. sandhangan panyigeg wanda yaiku sandhangan sing digunakake kanggo ngganteni aksara sigeg.

3. a) adhiku nulis layang kanggo Bulik

b) eyang nganggo jarik Sida Mukti

4. nguleg sambel nganggo lemper

5. nulis aksara Jawa

Brata menyang sekolah numpak pit onthel bareng bareng.

➡️ ꧋ꦧꦿꦠꦩꦼꦚꦁꦱꦼꦏꦺꦴꦭꦃꦤꦸꦩ꧀ꦥꦏ꧀ꦥꦶ

ꦠ꧀ꦲꦺꦴꦤ꧀ꦛꦺꦭ꧀ꦧꦉꦁꦧꦉꦁ꧉

Pembahasan

Nomer 1

Bapak tindak kanto = ꦧꦥꦏ꧀ꦠꦶꦤ꧀ꦢꦏ꧀ꦏꦤ꧀ꦠꦺꦴ

bapak, aksara Ba, aksara Pa, aksara Katindak, pasangan aksara Ta + wulu, aksara Na, pasangan aksara Da, aksara Kakanto, pasangan aksara La, aksara Na, pasangan aksara Ta + taling tarung

★ tembung kanto => kudune kurang sandhangan swara layar ( r ), dadi unine kantor ( ꧀ꦏꦤ꧀ꦠꦺꦴꦂ ).

Nomer 2

=> sandhangan panyigeg wanda yaiku sandhangan sing digunakake kanggo ngganteni aksara sigeg utawa huruf mati, kajaba aksara sigeg mau h, r, lan Ng.

=> sandhangan panyigeg wanda ana 4 ;

wignyan = paten H ➡️ ꦃlayar = paten R ➡️ ꦂcecak = paten ng ➡️ ꦁpangkon = saliyane paten H, R lan ng ➡️ ꧀

Nomer 3

a) adhiku nulis layang kanggo Bulik

> sandhangan swara :

wulu = i ➡️ Adhiku suku = u ➡️ nulis taling tarung = o ➡️ kanggo

> sandhangan panyigeg wanda :

cecak = Ng ➡️ kanggo pangkon = paten k ➡️ Bulik

Nomer 4

nguleg sambel nganggo lemper

nguleg sambel, aksara Nga + suku, aksara La + pepet, aksara Ga, pasangan aksara Sa, aksara Ma, pasangan aksara Ba + pepet, aksara Langanggo lemper, pasangan aksara Nga + cecak, aksara Ga + taling tarung, aksara La + taling, aksara Ma, pasangan aksara Pa + taling + layar.

Nomer 5

Brata menyang sekolah numpak pit onthel bareng bareng

Brata, aksara Ba + cakra, aksara Tamenyang sekolah, aksara Ma + pepet, aksara Nya + cecak, aksara Sa + pepet, aksara Ka + taling tarung, aksara La + wignyannumpak pit onthel, aksara Na + suku, aksara Ma, pasangan aksara Pa, aksara Ka, pasangan aksara Pa + wulu, aksara Ta, pasangan aksara Ha + taling tarung, aksara Na, pasangan aksara Tha + taling, aksara Labareng bareng, pasangan aksara Ba, aksara Pa cerak + cecak, aksara Ba, aksara Pa cerak + cecak.

4. Kapilih sing manggon ing ngarep menawa ora ana ya aksara mburine lan sakteruse.

Pembahasan

Nulis aksara jawa iku sing kudu dimangerteni yaiku pasangan lan sandhangane.

1. Balon biru = ꦧꦭꦺꦴꦤ꧀ꦧꦶꦫꦸ

balon = aksara Ba, aksara La + taling tarung, aksara Na (sigeg)biru = pasangan aksara Ba + wulu, aksara Ra + suku

2. masak tahu = ꦩꦱꦏ꧀ꦠꦲꦸ

masak = aksara Ma, aksara Sa, aksara Ka (sigeg)tahu = pasangan aksara Ta, aksara Ha + suku

3. tas lan topi = ꦠꦱ꧀ꦭꦤ꧀ꦠꦺꦴꦥꦶ

tas = aksara Ta, aksara Sa (sigeg)lan = pasangan aksara la, aksara Na ( sigeg)topi = pasangan aksara Ta + taling tarung, aksara Pa + wulu

4. jaket Abang = ꦗꦏꦺꦠ꧀ꦲꦧꦁ

jaket = aksara Ja, aksara Ka + taling, aksara Ta (sigeg)Abang = pasangan aksara Ha, aksara Ba + cecak

5. dolanan dakon = ꦢꦺꦴꦭꦤꦤ꧀ꦢꦏꦺꦴꦤ꧀

dolanan = aksara Da + taling tarung, aksara La, aksara Na, aksara Na (sigeg)dakon = pasangan aksara Da, aksara Ka + taling tarung, aksara Na + pangkon

38. Ini Gimana Yak?:? (Soal Kelas 5) Aksara Jawa Pliss ​Jawab Yang Bener Jangan Ngasal Aksara Jawa Dijadikan Bahasa Jawa Krama Inggil..


Ngartekake aksara jawa

Budi pekerti luhur

Budi pekerti luhur inggih punika sedaya solah bawa utawi tingkah aku (laku) ingkang sae saged dipun wastani morilipun manungsa, tumrap piyantun jawi budi pakerti luhur punika awujud tata krama, inggih punika sedaya pamicara lan tandang tanduk ingkang dipun tata manut kaliyan Unggah ungguh basa. Budi pekerti kedah dipunatosaken wonten ing sesrawungan ing bebrayan utawi masyarakat, ngormati tiyang sepuh, santun dhumatheng sasama, tresna asih dhumatheng ingkang langkung Mudha.

Budi pekerti bangsa kita ingkang kebak tata Krama punika dados cirikas ingkang kawentar ing Negari manca minangka warisanipun para leluhur.

__________________Pembahasan

Wacana wonten nginggil punika mboten ngginakaken aksara murda, aksara rekan utawi aksara swara.

Pangertenipun aksara jawa inggih punika aksara ingkang dipun serat kaliyan huruf jawi kuno.

Cacahipun aksara jawi, wonten kalehdasa (20) :

ha, na, ca,, ra, kada, ta, sa, wa,, lapa, dha, Ja, ya, nyama, ga, ba, tha, nga

Aksara Jawi ugi Kedah dipun paringi sandhangan aksara ingkang cacahipun wonten 3 :

sandhangan swara (wulu, suku, pepet, taling tarung, taling)sandhangan panyigeg Wanda (wignyan, layar, cecak, pangkon)sandhangan wyanjana (Cakra, Keret, pengkal)

Menawi Pangerten basa Krama Alus inggih punika basa ingkang tembung tembungipun ngginakaken basa Krama Alus. Dados biasanipun langkung sae.

Ginanipun kagem rembagan :

putra mring tiyang sepuhmurid mring guruwayah mring eyang

Tuladhanipun :

eyang gerah amargi bibar dhawah saking motorsampun kesesa kondur, nyare dhateng mriki kemawon
