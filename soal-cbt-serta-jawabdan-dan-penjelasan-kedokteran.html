11. tolong lah jawabdi kumpulkan plis​


Jawaban:

Pembahasan

Diketahui :

Suhu mula-mula = 26°C

Suhu diturunkan = -4°C

Ditanya :

Besar perubahan suhu?

Penyelesaian :

Menentukan perubahan suhu didalam ruangan

Perubahan suhu = suhu awal - suhu diturunkan

= 26°C - (-4°C)

= 26°C + 4°C

= 30°C

Jadi besar perubahan suhu di dalam ruangan tersebut adalah 30°C.

24. plss btu lagi CBT, helppp


Jawaban:

52 dm^2

Penjelasan dengan langkah-langkah:

Lp = 2 (pl + pt + lt)

Lp = 2 (4 × 2 + 4 × 3 + 2 × 3)

Lp = 2 (8 + 12 + 6)

Lp = 2 (26)

Lp = 52 dm^2

Semoga Membantu

# No Copas

# Saya kerjain serius

Diketahui = Panjang balok = 4 dm

= Lebar balok = 2 dm

= Tinggi balok = 3 dm

Ditanya = Luas permukaan balok?

Jawab :

Luas permukaan balok = 2 × ( p×l + p×t + l×t )

= 2 × ( 4×2 + 4×3 + 2×3 )

= 2 × ( 8 + 12 + 6 )

= 2 × 26

= 52 dm²

Jadi, luas permukaan balok adalah 52 dm²
