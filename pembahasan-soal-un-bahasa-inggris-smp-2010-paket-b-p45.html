36. UN 2009 PAKET A/B, UN 2010 PAKET B Akar-akar persamaan kuadrat x 2 + (a – 1)x + 2 = 0 adalah α dan β. Jika α = 2β dan a > 0 maka nilai a


x² + (a-1)x + 2 = 0 ; akar2 α dan β ; α = 2β

α.β = 2 ; subtitusi α = 2β
2β² = 2
  β² = 1
   β = 1 atau -1

saat β = 1 ; maka α = 2
α+β = -(a-1)
2+1 = -a+1
    a = -2 (ga cocok, karena a > 0)

saat β = -1 ; maka α = -2
α+β = -(a-1)
-2-1 = -a+1
    a = 4 (cocok!)
