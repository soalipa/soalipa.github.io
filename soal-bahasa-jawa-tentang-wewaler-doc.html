6. teks eksposisi wewaler "Ojo Nabuhi Pager Wayah Wengi" dalam bahasa jawa​


Ojo Nabuhi Pager Wayah Wengi

Wewaler iki luwih akrab dene kita ngendi- ngendi menawa kita loro kahanan. Yaiku waktu lan umur. Kita tak eling, sedino sing wis kedadeyan kok ra bakal dadi tenggang sing liyan. Ing dasare iki, kudu piyambak kang ora nganti- nganti utawa nglelakoni tugas, jangkepake nanging ora dibutuhake.

Salah sawijining wewaler sing mesti diwenehi perhatian yaiku "Ojo Nabuhi Pager Wayah Wengi". Ing basa Jawa, wewaler iki artine ora nebak karo yen kita ora boleh marang wong liyan kang ra cocok, utawa ora pengin tuku barang sing ora keprihatin. Wewaler iki nuduhake marang pangrasa kita, kudu bisa mlebu menyang ati lan nganggo akal sehat.

Wewaler "Ojo Nabuhi Pager Wayah Wengi" nuduhake marang pentingnya ngontrol pikiran kita, supaya kita bisa ngirim mungsuh, kekhilafan, lan sifat negatif liyane marang gedhene. Sing ngerti nanging ngarepake umur, kita kudu menghargai sapa sing ora miwiti kabeh. Takon, apa iya kita pengin nganggo wengi kita kanggo ngomong menyang sing kudu ditandurake?

Lan bab iki ora sing paling ora penting, sedino kita duwe paguyuban utawa jaringan sosial kang penting, wewaler iki nuduhake marang pentingnya ngontrol bisnis utawa kerja kita supaya ora nglawan aturan, lan ora seneng nggunakake cara sing ora benar.

Kesimpulane, wewaler "Ojo Nabuhi Pager Wayah Wengi" nuduhake marang pentingnya ngontrol tindakan kita, ngirim mungsuh, ngontrol pikiran kita, nglakoni tugas sing tepat, lan ngontrol bisnis utawa kerja kita. Mugi-mugi wewaler iki bisa ngrangsang kita kanggo ngrasani tindakan kita.

34. apa yang dimaksud DOC


Dalam teknologi komputasi, DOC atau doc (singkatan dari 'document' ) adalah sebuah ekstensi berkas untuk aplikasi pengolah kata yang terdapat di Microsoft Word Binary File Format.[4] Sejarah mengatakan bahwa ekstensi tersebut pernah digunakan pada teks biasa, khususnya program atau perangkat keras komputer di berbagai sistem operasi. Pada 1980an, WordPerfect menggunakan DOC sebagai format bawaan mereka. Kemudian, pada dasawarsa 1990an, Microsoftmemilih untuk menggunakan ekstensi DOC untuk format bawaaan Microsoft Word.
