Perpangkatan adalah sebuah bentuk operasi dalam ilmu matematika yang boleh digunakan apabila sebuah bilangan dikali dengan bilangan yang sama lebih dari satu kali. Pangkat sendiri merupakan banyaknya bilangan yang dikali dengan bilangan yang sama. Letak pangkat sendiri berada diatas bilangan yang dikalikan tersebut.

Bilangan berpangkat merupakan perkalian berulang suatu bilangan, dimana bilangan dapat berpangkat bulat positif, nol, maupun bulat negatif.

Contoh :

A⁴ = a × a × a × aA³ = a × a × aA² = a × a

Contoh Perpangkatan 2:

2² = ( 2 x 2 ) = 43² = ( 3 x 3 ) = 94² = ( 4 x 4 ) = 165² = ( 5 x 5 ) = 25

Contoh Perpangkatan 3:

2³ = ( 2 x 2 x 2 ) = 83³ = ( 3 x 3 x 3 ) = 274³ = ( 4 x 4 x 4 ) = 645³ = ( 5 x 5 x 5 ) = 125

Contoh Perpangkatan 4:

2⁴ = ( 2 x 2 x 2 x 2 ) = 163⁴ = ( 3 x 3 x 3 x 3 ) = 914⁴ = ( 4 x 4 x 4 x 4 ) = 2565⁴ = ( 5 x 5 x 5 x 5 ) = 625

Contoh Perpangkatan 5:

2⁵ = ( 2 x 2 x 2 x 2 x 2 ) = 323⁵ = ( 3 x 3 x 3 x 3 x 3 ) = 2734⁵ = ( 4 x 4 x 4 x 4 x 4 ) = 1.0245⁵ = ( 5 x 5 x 5 x 5 x 5 ) = 3.125

» Bilangan berpangkat merupakan perkalian berulang suatu bilangan (asli) dengan bilangan itu sendiri beberapa kali sebanyak yg ditunjukkan oleh bilangan (asli) itu.

» Sifat Sifat bilangan berpangkat

[tex]\boxed{ {a}^{m} \times {a}^{n} = {a}^{m + n} } \\ \boxed{ {a}^{m} \div {a}^{n} = {a}^{m - n} } \\ \boxed{( {a}^{m} {)}^{n} = {a}^{m \times n} } \\ \boxed{(ab {)}^{n} = {a}^{n} \times {b}^{n} } \\ \boxed{( \frac{a}{b} {)}^{n} = \frac{ {a}^{n} }{ {b}^{n} } }[/tex]

» Jenis jenis bilangan berpangkat

Bilangan berpangkat Positif yaitu Pangkat nya positif, tidak ada min ( - )Bilangan berpangkat Negatif yaitu Pangkat nya ada min ( - )Bilangan berpangkat nol ( 0 ) yaitu pangkat nya tidak ada, atau bisa dibilang 1

» Bentuk akar Merupakan bilangan yg menunjukkan pangkat dari bilangan tersebut dibagi oleh indeks yang terdapat dalam akar.

1 Bu titik mempunyai toko pakaian. Bu titik membeli 12 lusin celana dan 20 kodi baju untuk melengkapi tokonya. Berapa buah celana dan baju yang dibeli bu titik ?

Jawab :

122 + 202 = 144 + 400 = 544

Jadi ada 544 buah celana dan baju yang dibeli bu titik.

2. Ibu membeli karpet berbentuk persegi dengan luas 62.500 cm2. Berapa sentimeter panjang sisi karpet ?

Jawab :

√62500 = 250

Jadi panjang sisi karpet 250 cm.

3. Paman membeli 144 buah buku untuk dibagikan kepada anak-anak panti asuhan. Berapa lusin buku yang dibeli paman ?

Jawab :

√144 = 12

Jadi ada 12 lusin buku yang dibeli paman.

4. Sebuah percetakan membeli 500 rim kertas. Berapa lembar kertas yang dibeli perusahaan tersebut ?

Jawab :

5002 = 250000

Jadi ada 250000 lembar kertas yang dibeli perusahaan.

5. Pak toni membeli sebidang tanah berbentuk persegi dengan harga Rp3.125.000,00 untuk membangun warung makan. Apabila tiap 1 m2 harganya Rp125.000,00, berapakah panjang sisi persegi tersebut ?

Jawab :

3125000 : 125000 = 25

√25 = 5

Jadi panjang sisi persegi adalah 5 m.

6. Pak yanto mempunyai dua petak tanah masing-masing berbentuk persegi. Panjang sisi petak tanah yang pertama setengah panjang sisi petak tanah yang kedua. Luas petak tanah yang kedua 484 m2. Tentukan luas petak tanah yang pertama ?

Jawab :

√484 = 22

Jadi luas petak tanah pertama adalah 22 m2.

Jawab :

√100 = 10

Jadi ada 10 kilogram beras yang dapat diangkut truk.

8. Bu titik seorang pembuat kue. Bu titik mendapat pesanan 24 kotak kue donat. Setiap kotak berisi 2 lusin donat. Berapa buah donat yang harus dibuat bu titik ?

Jawab :

242 × 22 = 576 × 4 = 2304

Jadi ada 2304 buah donat yang harus dibuat bu titik.

[tex]{\bold{Bilangan\:Berpangkat}}[/tex]merupakan bentuk sederhana dari perkalian berulang. Bilangan Berpangkat dinyatakan sebagai bilangan yang dikalikan dengan bilangan itu sendiri secara berulang.
