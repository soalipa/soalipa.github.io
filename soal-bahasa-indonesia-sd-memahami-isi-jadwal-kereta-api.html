25. Sebuah kereta api meninggalkan stasiun 1 jam sebelum waktu yang dijadwalkan. Masinisnya menurunkan kecepatannya 20km/h. Di stasiun berikutnya, sejauh 120 km, kereta itu sampai tepat waktu sesuai jadwal. Kecepatan awal kereta api sebelum dilambatkan adalah (km/h)


Kecepatan awal = v1
waktu tempuh dg v1 = t1

kecepatan yg dipake = v2 = 20 km/h
t2 = (t1 + 1) jam

s = 120 km
s1 = v1 . t1 = 120 km

s2 = v2 × t2 = 120
120 = 20 × (t1 + 1)
t1 + 1 = 120/20
t1 = 6 - 1
t1 = 5 jam

S1 = v1 × t1 = 120
v1 = 120/t1
v1 = 120/5
v1 = 24 km/h

Jadi kecepatan awal KA sebelum dilambatkan = v1 = 24 km/h
