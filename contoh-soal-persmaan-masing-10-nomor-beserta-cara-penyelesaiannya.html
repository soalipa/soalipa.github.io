30. Himpunan penyelesaian persmaan cos 2x +7 sinx -4=0


Mapel : Matematika

Materi : Trigonometri

Soal

Himpunan Penyelesaian Persamaan

cos(2x) + 7sin(x) - 4 = 0 adalah ?

Pembahasan

cos(2x) + 7sin(x) - 4 = 0

1 - 2sin²(x) + 7sin(x) - 4 = 0

-2sin²(x) + 7sin(x) - 3 = 0 (dikali -1)

2sin²(x) - 7sin(x) + 3 = 0

misalkan sin(x) = p, maka

2p² - 7p + 3 = 0

(2p - 1)(p - 3) = 0

p = ½ atau p = 3 (TM)

p = ½

sin(x) = sin30°

• mencari x dengan rumus α + k(360)

- Untuk k = 0

x = 30°

- Untuk k = 1

x = 30° + 360°

x = 390° (dan seterusnya)

• mencari x dengan rumus (180 - α) + k(360)

- Untuk k = 0

x = (180 - 30) + 0

x = 150°

- Untuk k = 1

x = (180 - 30) + 360

x = 150 + 360

x = 510° (dan seterusnya)

maka HP = {30°, 150°, 390°, 510°, ....}
