Belog
Disebuah desa di Bali hidup seorang anak yang konyol tapi bahagia yang bernama Belog. 'Belog' berarti 'Bodoh' dalam bahasa Bali, dan Belog terkenal di desanya karena hal-hal bodoh yang selalu ia lakukan. Warga di desa itu tidak pernah lelah menceritakan satu sama lain cerita tentang perbuatan Belog yang berpikiran simpel. Salah satu cerita favorit seperti berikut.
Satu hari ibu Belog tidak sengaja menjatuhkan korek api terakhirnya kedalam genangan air. korek api tersebut menjadi basah dan tidak bisa dinyalakan. Dia sekarang butuh korek api untuk menyalakan kompor, jadi dia memanggil putranya dan berkata, 'Belog tolong pergi ke toko dan beli beberapa korek api. Korek api yang kupunya basah, dan aku tidak bisa menyalakan api untuk memasak makanan kita.' dia memberikan Belog uang dan berkata " sekarang pastikan bahwa korek api yang kau beli kering dan dapat dinyalakan dengan mudah!"
Lalu Belog mengambil uang untuk membeli korek api dari ibunya, dan dengan bahagia berjalan ke toko. Dia masuk kedalam dan meminta sekotak korek api kepada pemilik toko. Lalu Belog dengan hati-hati menghitung uangnya dan membayar korek api tersebut lalu pulang kerumahnya.
Namun ketika dia diluar toko dia mendapat sebuah ide. Dia membuka kotak itu dan melihat kedalamnya. "korek api ini nampak bagus menurutku", pikirnya " tapi bagaimana bisa aku yakin? sebaiknya mencoba satu". Belog mengeluarkan satu dan menggeseknya. korek api itu menyala dengan mudah.
"Ah, korek api ini bagus," katanya "namun bagaimana dengan yang lainnya?" jadi Belog mengeluarkan korek api tersebut satu demi satu untuk untuk mencoba apakah korek api tersebut bagus lalu dia meletakkan kembali korek api tersebut kedalam kotak " ini sekotak korek api yang bagus" Belog berkata kepada dirinya lalu kembali kerumah.
" ini ibu" kata Belog, dengan senyum yang jujur dan konyol di mukanya "aku membeli sekotak korek api yang bagus. semuanya menyala, aku telah mencoba semuanya!" lalu Belog memberikan ibunya sekotak korek api yang telah dipakai dan dibakar
Ibunya yang malang. apa yang bisa ia katakan? lagipula Belog adalah anak satu-satunya, dan dia sangat mencintainya.

15. soal Mengetahui Sejarah Dan Struktur Blog​


Penjelasan:

Sejarah Blog:

Blog, singkatan dari "weblog", pertama kali muncul pada tahun 1990-an sebagai bentuk jurnal online yang diupdate secara teratur. Pada awalnya, blog digunakan sebagai platform untuk berbagi pemikiran, pengalaman, dan informasi pribadi. Blog awalnya hanya berupa teks, tetapi seiring waktu, mereka berkembang menjadi platform yang lebih interaktif dengan penambahan gambar, video, dan fitur sosial.

Pada tahun 1999, platform blogging populer seperti Blogger dan LiveJournal diluncurkan, yang memungkinkan pengguna untuk membuat dan mengelola blog mereka sendiri dengan mudah. Pada tahun 2003, WordPress, platform blogging yang sangat populer, diluncurkan, yang memungkinkan pengguna untuk membuat blog dengan tampilan yang lebih profesional dan fitur yang lebih canggih.

Seiring dengan perkembangan teknologi dan popularitas internet, blog menjadi semakin populer di kalangan individu, perusahaan, dan organisasi. Blog digunakan untuk berbagai tujuan, termasuk jurnal pribadi, jurnalisme warga, pemasaran konten, dan platform pendidikan.

Struktur Blog:

Struktur blog terdiri dari beberapa elemen penting yang membentuk tampilan dan fungsionalitas blog. Berikut adalah beberapa elemen struktur blog yang umum:

1. Judul dan Header: Bagian atas blog yang mencakup judul blog dan mungkin juga logo atau gambar header yang unik.

2. Menu Navigasi: Biasanya terletak di bagian atas atau samping blog, menu navigasi berisi tautan ke halaman-halaman penting seperti beranda, tentang kami, kontak, dan kategori blog.

3. Konten Utama: Ini adalah bagian utama dari blog yang berisi postingan terbaru atau artikel yang diurutkan berdasarkan tanggal atau kategori. Konten utama biasanya terdiri dari judul, isi teks, gambar, video, dan tautan terkait.

4. Sidebar: Terletak di samping atau di bawah konten utama, sidebar berisi elemen tambahan seperti daftar arsip, kategori, tautan terkait, kotak pencarian, widget media sosial, dan iklan.

5. Footer: Bagian bawah blog yang berisi informasi tambahan seperti tautan ke halaman-halaman penting, hak cipta, dan informasi kontak.

6. Komentar: Fitur komentar memungkinkan pembaca untuk berinteraksi dengan pemilik blog dan pengunjung lainnya dengan memberikan tanggapan atau pertanyaan di bawah postingan.

7. Kategori dan Label: Blog sering menggunakan sistem kategori dan label untuk mengorganisir dan mengindeks postingan berdasarkan topik atau tema tertentu. Ini membantu pembaca menemukan konten yang relevan dengan minat mereka.

8. Arsip: Halaman arsip menyediakan daftar postingan blog yang diurutkan berdasarkan bulan dan tahun, memungkinkan pembaca untuk menjelajahi konten lama.

9. Halaman Statis: Selain postingan blog, blog juga dapat memiliki halaman statis seperti halaman tentang kami, kontak, atau kebijakan privasi.

10. Responsif dan Desain: Blog modern sering dirancang agar responsif, artinya tampilan dan tata letaknya dapat menyesuaikan dengan perangkat yang digunakan oleh pengunjung, seperti komputer, tablet, atau ponsel pintar.

Struktur blog dapat bervariasi tergantung pada platform blogging yang digunakan dan preferensi pemilik blog. Namun, elemen-elemen di atas umumnya ada dalam sebagian besar blog.

Pengertian Upload

Upload dapat juga disebut dengan Unggah, Upload merupakan kebalikan dari Download. Pengertian Upload adalah proses mengirim file yang tersimpan dari komputer lokal ke komputer sistem jaringan Internet. Secara umum upload dapat diartikan sebagai proses transmisi data dari pemakai komputer/komputer client ke jaringan internet.

Atau upload adalah mengunggah file atau data (bisa berupa dokumen, foto, audio, video) dari komputer pribadi ke sebuah sistem server, dan file atau data tersebut akan dipublikasikan di internet, sehingga data yang telah diupload tersebut dapat dilihat, di download (diambil) oleh orang lain.

Contoh Upload

Tujuan Upload

Tujuan dari upload adalah supaya pengguna internet yang berada di dunia bisa melihat, memanfaatkan atau menggunakan file yang kita upload. Hal tersebut dapat kita contohkan orang lain yang dapat menikmati karya tulisan dalam bentuk data atau file yang kita pajang pada blog, media sosial atau media lainnya. Selain itu tujuan upload yaitu untuk menyimpan data pada sebuah server agar apabila kita kehilangan data tersebut pada komputer kita, kita bisa mencarinya kembali pada server tersebut.

Kembali ke Menu Pembahasan ↑

Manfaat Upload

Manfaat upload file yaitu kita bisa menghemat memori yang diperlukan untuk menyimpan sebuah file karena file kita pindahkan dengan cara upload ke dalam sebuah server yang menyediakan penyimpanan data atau file. Selain itu file yang telah kita upload sewaktu-waktu bisa kita ambil lagi dengan cara mendownloadnya. Manfaat lainnya yaitu kita tidak perlu khawatir jika file yang kita simpan pada komputer atau flash disk atau penyimpanan lain terhapus/ rusak karena file yang kita upload tersebut akan lebih aman dan bisa kita dowload sewaktu-waktu.

Kembali ke Menu Pembahasan ↑

Cara Upload File di Internet

Apabila kita ingin mengupload, terlebih dahulu kita hubungkan perangkat komputer atau handphone ke internet, kemudian pilih penyimpanan server yang kamu sukai, seperti google drive, drop box, 4shared, onedrive, mediafire, media sosial dan masih banyak yang lainnya. Setelah memilih server yang kamu sukai, kamu harus membuat akun dengan menggunakan email kamu. Setelah kamu mempunyai akun, kamu dapat mengupload file ke server tersebut.
