4. Diketahui :

U3 = 7

U10 = 35

Tentukan :

*Rumus suku ke - n

*Suku ke 20

pembahasan

U3 = a + ( 3-1 ) b = 7

= a + 2b = 7 ( persamaan ke 1 )

U10 = a + (10-1) b = 35

= a + 9b = 35 (persamaan ke 2)

Metode Eliminasi

a + 2b = 7

a + 9b = 35

____________ -

-7b = -28

b = 4

Metode Subtitusi

a + 2b = 7

a + 2(4) = 7

a + 8 = 7

a = 7 - 8

a = -1

Maka

Rumus Suku ke Un :

Un = -1 + ( n-1 ) 4

= -1 + 4n - 4

Un = 4n - 5

Suku ke 20 :

U20 = -1 + ( 20 - 1 ) 4

= -1 + (19)4

= -1 + 76

= 75

Misalkan a=jumlah soal salah dan b=jumlah soal benar.

2b - a = 70

b + a = 44 (2b + 2a = 88)

Gunakan metode eliminasi:

2b - a = 70

2b + 2a = 88 _

-3a = -18

a = 6

Jadi, jumlah soal yang dijawab salah ada 6.
