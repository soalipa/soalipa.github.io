Fungsi HTML

HTML (HyperText Markup Language) adalah suatu bahasa yang menggunakan tanda-tanda tertentu (tag) untuk menyatakan kode-kode yang harus ditafsirkan oleh browser agar halaman tersebut dapat ditampilkan secara benar.

Secara umum, fungsi HTML adalah untuk mengelola serangkaian data dan informasi sehingga suatu dokumen dapat diakses dan ditampilkan di Internet melalui layanan web.

Fungsi HTML yang lebih spesifik yaitu :

Membuat halaman web.

Menampilkan berbagai informasi di dalam sebuah browser Internet.

Membuat link menuju halaman web lain dengan kode tertentu (hypertext).

9. Mengapa HTML bukan termasuk bahasa pemrograman?


Jawaban:

HTML adalah singkatan dari Hypertext Markup Language. Pengertian HTML sering kali dikaitkan dengan bahasa pemrograman. Akan tetapi, sejatinya, HTML bukan sebuah bahasa pemrograman (programming languange), melainkan sebuah markup languange.

Jadi HTML yang merupakan bahasa markup tidak termasuk bahasa pemrograman karena tidak memiliki tag atau kode yang berfungi untuk menyelesaikan masalah. HTML hanya berfungsi memberikan tampilan luar bagi bahasa pemrograman web seperti php dan JavaScript, bersama dengan CSS.

30. ciri utama html tolong jawab


Ciri utama html adalah terdapatnya tag dan elemen html.

HTML atau Hypertext Markup Language adalah salah satu format yang biasa dipakai dalam pembuatan dokumen dan aplikasi di halaman web. Disebut sebagai Markup Language karena dalam text HTML mengandung tag (kode yang digunakan untuk me-mark up teks ASCII menjadi file HTML) tertentu yang digunakan untuk menentukan tampilan suatu teks dan tingkat urgensi dari teks tersebut dalam suatu dokumen.

Ciri - ciri utama dokumen html adalah sebagai berikut.

1. Terdapat Tag HTML

a. Tag ditandai dengan “<” dan “>”.

b. Dalam penulisan kode HTML harus diawali dengan <tag_pembuka> dan diakhiri dengan

</tag_penutup>.

c. Tag HTML tidak ‘case sensitive’, <b> sama dengan <B>.

d. Jika dalam suatu ag ada tag lagi maka penulisan tag akhir harus berurutan, misal <b><i> ...

e. Huruf Tebal Miring </i></b>.

2. Terdapat ELEMENT HTML, yang terdiri dari 2 bagian :

a. HEAD merupakan kepala dokumen dari HTML, yang berfungsi untuk memberikan informasi tentang dokumen.

b. BODY merupakan tubuh dari HTML, yang berfungsi untuk menentukan bagaimana isi suatu dokumen yang akan ditampilkan pada web browser.

31. Perintah html untuk menentukan, tebal garis table adalah


Hypertext Markup Language atau yang lebih sering dikenal dengan HTML adalah bahasa sederhana yang dipergunakan di dalam pembuatan halaman suatu website. Dalam operasional penggunaan html, perintah html yang digunakan untuk menentukan ketebalan garis table adalah Table Border.

Pembahasan

Di era digital saat ini, pembuatan situs web dan aplikasi seluler telah menjadi pusat tren untuk  meningkatkan promosi dan meluncurkan produk dan layanan secara efektif. Oleh karena itu, bekerja sebagai programmer adalah pilihan yang paling diinginkan di industri ini. Pengembang web juga harus mahir dalam bahasa markup khusus untuk membantu  struktur halaman situs web, desain HTML.

HTML atau bahasa markup hypertext adalah bahasa markup yang digunakan untuk menyusun halaman web untuk ditampilkan di browser web. Oleh karena itu, HTML dapat digunakan sebagai landasan pertama untuk membangun kerangka website terstruktur sebelum membahas  tampilan desain dan aspek fungsional.

Atribut border, cellpadding, dan cellpacing digunakan untuk mengubah tampilan tabel, terutama yang berkaitan dengan batas dan spasi antar sel tabel. Atribut border digunakan untuk mengatur ketebalan border tabel. Jika atribut ini tidak ditentukan, browser web  akan menampilkan tabel tanpa bingkai. Nilai  atribut ini adalah angka yang diukur dalam piksel. Jika Anda menentukan batas nilai = "2", browser web  menampilkan batas 2 piksel di bagian atas, bawah, kiri, dan kanan tabel.
