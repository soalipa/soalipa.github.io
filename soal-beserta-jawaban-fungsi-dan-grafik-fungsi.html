9. Soal Matematika : Gambarkan grafik fungsi kuadrat berikut :A. F(x) = x² + 2x-3B. F(x) = -x²+4x+12Jawab beserta langkah-langkahnya dan gambar grafiknya​


Gambar grafik sudah saya lampirkan, langkah langkahnya adalah :

Pertama, tentukan dulu titik titik stasioner, atau titik titik yang membentuk parabola sesuai grafiknya. Sebenarnya titik stasioner ini ada banyak jenisnya, tetapi untuk fungsi kuadrat (pangkat dua), mungkin nilai yang bisa kita jadikan alat pembuat grafik hanya :

Naik, turunAkar akartitik Ekstrim

Bagian A

f(x) = x² + 2x -3 untuk

f(x) = ax² + bx + c diperoleh :

a = 1

b = 2

c = -3

maka karena a = 1 yaitu a > 0 artinya dari titik Extreme, tarik dua garis keatas sampai akar akarnya dan membentuk kurva.

untuk menentukan akar akarnya, faktorkan saja dan ubah fungsi kuadrat ke persamaan kuadrat ;

x² + 2x -3 = 0

(x + 3) (x -1) = 0

x = -3 atau x = 1

maka akar akarnya adalah (-3, 0) dan (1, 0)

dan titik extrim cukup gunakan turunan, yaitu :

f(x) = x² + 2x -3

f'(x) = 2x + 2 = 0

2x + 2 = 0

2x = -2

x = -1

f(-1) = (-1)² + 2(-1) -3

f(-1) = 1 -2 -3

f(-1) = -4

maka titik extrimnya adalah (-1, -4)

Jadi langkah singkat membuat grafik f(x) adalah dari titik (-1, -4), tarik dua garis keatas masing masing sampai (-3, 0) dan (1, 0) dan membentuk kurva.

Bagian B

f(x) = -x² + 4x + 12 untuk

f(x) = ax² + bx + c diperoleh :

a = -1

b = 4

c = 12

maka karena a = -1 yaitu a < 0 artinya dari titik Extreme, tarik dua garis kebawah sampai akar akarnya dan membentuk kurva.

untuk menentukan akar akarnya, faktorkan saja dan ubah fungsi kuadrat ke persamaan kuadrat ;

-x² + 4x + 12 = 0

x² -4x -12 = 0

(x + 2)(x -6) = 0

x = -2 atau x = 6

maka akar akarnya adalah (-2, 0) dan (6, 0)

dan titik extrim cukup gunakan turunan, yaitu :

f(x) = -x² + 4x + 12

f'(x) = -2x + 4 = 0

-2x = -4

x = 2

f(2) = -(2)² + 4(2) + 12

f(2) = -4 + 8 + 12

f(2) = 16

maka titik extrimnya adalah (2, 16)

Jadi langkah singkat membuat grafik f(x) adalah dari titik (2, 16), tarik dua garis keatas masing masing sampai (-2, 0) dan (6, 0) dan membentuk kurva.

11. Tentukan Asimtot datar dan Asimtot tegak beserta gambar grafik nya dari fungsi berikutb. F(x) = 8/8-2 -4mohon di jawab ya kak soal matematika peminatan kelas 12 pakai cara Terima kasih telah membantu saya ​


Jawab:

Dalam kasus fungsi \(F(x) = \frac{8}{8 - 2x - 4}\), terdapat asimtot datar pada \(y = 0\) dan asimtot tegak pada \(x = 2\). Goresan grafiknya mendekati asimtot datar \(y = 0\) saat \(x\) mendekati tak hingga positif atau negatif, dan ada sebuah vertikal asimtot di \(x = 2\).

Asimtot vertikal pada x = 2 dan asimtot horizontal pada y = 0. Gambar grafiknya sulit untuk ditampilkan dalam format teks.

Penjelasan dengan langkah-langkah:

Untuk menentukan asimtot datar dan asimtot tegak dari fungsi F(x) = 8/(8 - 2x - 4), pertama-tama kita harus mencari di mana fungsi ini memiliki asimtot dan apa jenis asimtotnya. Asimtot adalah garis vertikal atau horizontal yang mendekati grafik fungsi tanpa pernah memotongnya.

Mari kita mulai dengan mencari asimtot horizontal dan vertikal:

**Asimtot Vertikal:**

Asimtot vertikal terjadi ketika penyebut dari fungsi mendekati nol. Dalam hal ini, penyebut adalah 8 - 2x - 4. Kita akan mencari nilai x ketika penyebut ini sama dengan nol:

8 - 2x - 4 = 0

Kemudian, kita akan mencari akar dari persamaan ini:

2x = 8 - 4

2x = 4

x = 2

Jadi, terdapat asimtot vertikal di x = 2.

**Asimtot Horizontal:**

Asimtot horizontal terjadi ketika nilai fungsi mendekati suatu nilai tetap saat x mendekati tak hingga positif atau negatif. Untuk mencari asimtot horizontal, kita perhatikan bahwa ketika x mendekati tak hingga positif atau negatif, penyebutnya mendekati nol (8 - 2x - 4 mendekati 0).

Sebagai x mendekati tak hingga positif:

lim (x -> ∞) [8/(8 - 2x - 4)] = 8/(-∞) = 0

Sebagai x mendekati tak hingga negatif:

lim (x -> -∞) [8/(8 - 2x - 4)] = 8/(∞) = 0

Jadi, terdapat asimtot horizontal di y = 0.

Sekarang mari kita lihat gambar grafiknya:

```

        |

        |             *

        |          *  

        |       *  

---------|-----------------------------------

        |  

        |  

```

- Garis vertikal pada x = 2 adalah asimtot vertikal.

- Garis horizontal pada y = 0 adalah asimtot horizontal.

Grafik fungsi ini mendekati asimtot-asimtot ini saat x mendekati tak hingga positif atau negatif dan saat x mendekati 2.
