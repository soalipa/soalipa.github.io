4. Mencari rumus beserta contoh soalnya pada cermin datar, cekung, dan cembung​


Jawaban:

Tentu! Berikut ini adalah rumus dan contoh soal untuk cermin datar, cekung, dan cembung:

1. Cermin Datar:

- Rumus: s = -s'

Keterangan:

- s = jarak objek ke cermin (positif jika objek berada di depan cermin, negatif jika objek berada di belakang cermin)

- s' = jarak bayangan ke cermin (positif jika bayangan ada di depan cermin, negatif jika bayangan ada di belakang cermin)

Contoh Soal: Sebuah objek ditempatkan 20 cm di depan cermin datar. Tentukan posisi dan sifat bayangan yang terbentuk jika menggunakan cermin datar.

2. Cermin Cekung:

- Rumus: 1/f = 1/s + 1/s'

Keterangan:

- f = jarak fokus cermin (positif untuk cermin cekung)

- s = jarak objek ke cermin (positif jika objek berada di depan cermin, negatif jika objek berada di belakang cermin)

- s' = jarak bayangan ke cermin (positif jika bayangan ada di depan cermin, negatif jika bayangan ada di belakang cermin)

Contoh Soal: Sebuah objek ditempatkan 30 cm di depan cermin cekung dengan jarak fokus 15 cm. Hitunglah posisi dan sifat bayangan yang terbentuk menggunakan cermin cekung.

3. Cermin Cembung:

- Rumus: 1/f = 1/s - 1/s'

Keterangan:

- f = jarak fokus cermin (negatif untuk cermin cembung)

- s = jarak objek ke cermin (positif jika objek berada di depan cermin, negatif jika objek berada di belakang cermin)

- s' = jarak bayangan ke cermin (positif jika bayangan ada di depan cermin, negatif jika bayangan ada di belakang cermin)

Contoh Soal: Sebuah objek ditempatkan 40 cm di depan cermin cembung dengan jarak fokus -20 cm. Tentukan posisi dan sifat bayangan yang terbentuk menggunakan cermin cembung.

Perhatikan bahwa dalam contoh soal di atas, nilai jarak objek (s) dan jarak bayangan (s') diberikan dengan tanda positif atau negatif sesuai dengan konvensi yang digunakan dalam fisika cermin.

8. fungsi y= 2x³-x²+4x+6 adalah... Tentukan : titik belok nya, daerah fungsi cekung ke atas dan kebawah ?


Diketahui: fungsi y = 2x³ - x² + 4x + 6

Ditanya: titik belok, daerah fungsi cekung ke atas dan ke bawah dari fungsi berikut:

Penyelesaian:

Untuk menentukan interval x yang membuat daerah fungsi tersebut cekung ke atas atau ke bawah, terlebih dulu kita tentukan turunan pertama dan turunan kedua dari fungsi tersebut:

F(x) = y =  2x³ - x² + 4x + 6

F’(x) = y’ = 6x² - 2x + 4

F”(x) = y” = 12x – 2

 

Jika f(x) cekung ke atas maka f”(x) > 0

F”(x) > 0

12x – 2 > 0

12x > 2

X > 1/6

 

Jika f(x) cekung ke bawah maka f”(x) < 0

F”(x) < 0

12x – 2 < 0

12x < 2

X < 1/6

 

Jadi, daerah fungsi f(x) = 2x³ - x² + 4x + 6 cekung ke atas pada interval x > 1/6 dan daerah fungsi f(x) = 2x³ - x² + 4x + 6 cekung ke bawah pada interval x < 1/6.

 

Selanjutnya kita tentukan titik belok fungsi tersebut:

Terlebih dulu kita cari turunan kedua fungsi, karena di awal kita sudah tentukan, maka:

F”(x) = y” = 12x – 2

Kemudian kita definisikan f”(x) = 0, maka

12x – 2 = 0

12x = 2

X = 1/6

Untuk x = 1/6, kita cari hasil f(x), maka diperoleh

F(x) = 2x³ - x² + 4x + 6

F(1/6) = 2(1/6)³ - (1/6)² + 4(1/6) + 6

F(1/6) = 359/54

 

Titik (1/6, 359/54) adalah titik belok fungsi f.

Jadi, titik belok fungsi f(x) = 2x³ - x² + 4x + 6 adalah ([tex] \frac{1}{6} [/tex], [tex] \frac{359}{54} [/tex])

[tex] \begin{align} f^{\prime \prime}(x) &= 0 \\ \frac{\mathrm{d}}{\mathrm{d}x} \left[ 2x - 2c \right] &= 0 \\ 2 &\ne 0 \end{align} [/tex]

Disini tidak ada nilai [tex] x [/tex], maka tidak terdapat titik belok.

(d) Selang kecekungan (cekung ke atas)

Ini mirip dengan cara sebelumnya, namun turunan fungsi kedua harus lebih kecil daripada nol.

[tex] f^{\prime \prime}(x) < 0 \Leftrightarrow 2 \nless 0 [/tex]

Tidak ada fungsi yang cekung ke atas di [tex] x \in \mathbb{R} [/tex].

[tex] f^{\prime \prime}(x) > 0 \Leftrightarrow 2 > 0 [/tex]

Yang berarti, fungsi ini cekung ke bawah untuk [tex] x \in \mathbb{R} [/tex]
