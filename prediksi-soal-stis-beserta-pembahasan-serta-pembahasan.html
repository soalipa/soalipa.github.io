21. Tolong dibantu kak.. Usm stis


∫dx/(e^x + 1)

misal, e^x + 1 = u
e^x = u - 1
e^x dx = du
dx = du/(e^x)
dx = du/(u - 1)

∫dx/(e^x + 1) = ∫(du/(u - 1))/u
= ∫du/(u(u - 1))
= ln|u - 1| - ln|u| + C
= ln |(u - 1)/u| + C
= ln|e^x/(e^x + 1)| + C
= ln|(e^x + 1)/(e^x)|^(-1) + C
= -ln|(e^x + 1)/(e^x)| + C
= -ln|1 + 1/e^x| + C
= -ln|1 + e^(-x)| + C

Jawaban tidak ada dalam opsi.

31. Separuh peserta ujian PMB STIS gagal karena tidak lulus pada tahap ujian pertama. 300 orang peserta ujian lainnya gagal setelah tes psikotest, sehingga tinggal 2/5 dari seluruh pesertaujian PMB STIS. Jumlah peserta ujian PMB STIS seluruhnya adalah ... A. 1200 B. 1500 C. 2700 D. 3000 E. 5000


Jawaban:

D. 3000

Penjelasan:

Misal jumlah seluruh peserta PMB STIS seluruhnya = x.

x - (0.5x + 300) = (2/5) x

-(0.5x + 300) = (2/5) x - x

-(0.5x + 300) = -(3/5) x

(0.5x + 300) = (3/5) x

(0.5x + 300) = 0.6x

0.1x = 300

x = 3000

Jadi, jumlah seluruh peserta PMB STIS sebanyak 3000 peserta.
