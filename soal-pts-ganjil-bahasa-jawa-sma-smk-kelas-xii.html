7. Tolong dibantu soal kelas XII SMA tentang himpunan penyelesaian. Tks


2 sin (2x + phi) - V3 = 0
2sin (2x + phi) = V3
sin (2x + phi) = (V3)/2
sin (2x + phi) = 60°
sin (2x + phi) = sin 60°
2x + phi = 60° + k.360
2x = 60 - phi + k.360
2x = -120 + k.360
x = -60 + k.180
untuk k = 0, => x = -60
untuk k = 1, => x = 120
untuk k = 2 => x = 300
HP = { 120°, 300° }

30. soal pts bahasa jepang kelas 12​


EACD

Pembahasan

"Hobi" dalam Bahasa Jepang disebut dengan 趣味 (Shumi). Untuk menyatakan hobi kita dalam Bahasa Jepang, dapat menggunakan pola kalimat berikut.

=> Watashi no shumi wa [...] desu.

=> Hobi saya [...].

*'Watashi no' bisa dihilangkan jika lawan bicara sudah mengetahui bahwa subjeknya adalah kita sendiri.

Contoh-contoh hobi dalam Bahasa Jepang antara lain adalah sebagai berikut.

Anime o miru koto (Menonton anime)Ongaku o kiku koto (Mendengar musik)E o kaku koto (Menggambar)Oyogu koto (Berenang)Benkyou suru koto (Belajar)Sakkaa o suru koto (Bermain bola)Shashin o toru koto (Mengambil foto)Gitaa o hiku koto (Bermain gitar)Kosupuree o suru koto (Meng-cosplay)Okashi o tsukuru koto (Membuat kue)Manga o kaku koto (Menggambar komik)dll...

Hobi-hobi yang saya sebutkan di atas, bisa dimasukkan ke bagian "[...]" di pola kalimat yang sudah saya berikan sebelumnya.

Contohnya sebagai berikut.

Watashi no shumi wa anime o miru koto desu. (Hobi saya menonton anime)Watashi no shumi wa ongaku o kiku koto desu. (Hobi saya mendengar musik)dll...

Untuk membuat kalimat tanya tentang hobi, dapat menggunakan pola kalimat berikut.

=> [Subjek] no shumi wa nan desuka.

=> Apa hobi [Subjek]?

*'[Subjek] no' bisa dihilangkan jika lawan bicara sudah mengetahui siapa subjeknya.

40. imajinasi halaman 12 di buku seni budaya SMA/SMK kelas XII​


Suatu imajinasi yang dapat dilihat dalam seni adalah imajinasi bahwa suatu karya ini memilki jiwa dan historinya tersendiri. Contohnya dalam melukis suatu figur kita membayangkan bahwa figur tersebut nyata dan memilki karakter sendirinya.

Pembahasan:

Imajinasi adalah daya pikir untuk membayangkan peristiwa (fantasi) atau membuat gambar (lukisan, karangan, dll) berdasarkan kenyataan dan pengalaman sendiri secara umum.

Istilah ini digunakan secara profesional dalam psikologi sebagai proses merekonstruksi persepsi objek yang sebelumnya diberikan persepsi pemahaman. Karena penggunaan istilah tersebut bertentangan dengan  bahasa sehari-hari, beberapa psikolog lebih suka berbicara tentang "ilustrasi" atau "citra" atau  reproduksi daripada imajinasi "produktif" atau "konstruktif".

Gambar dipahami sebagai sesuatu yang terlihat di "mata pikiran". ". Satu hipotesis tentang evolusi imajinasi manusia adalah bahwa ia menggunakan simulasi jiwa untuk memungkinkan setiap makhluk  sadar  memecahkan masalah individu.
