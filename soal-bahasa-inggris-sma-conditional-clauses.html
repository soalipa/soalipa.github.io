Subject : English


Category : Conditional Sentence


Level : SHS XII

Code : 12.5

Keyword : If



---------

Question :


We .... the seven seas if we had a yacht.

----------

Kalimat di atas menggunakan 2nd Conditional (S + future in the past IF simple past) karena if clausenya menggunakan past simple (S + v₂).

Dalam 2nd conditional main clausenya menggunakan future in the past (S + would + v₁)

----------

Answer :

We would sail the seven seas if we had a yacht.

Rumus First Conditional: If + (simple present tense), (simple future tense).

conditional sentence type 1 = mungkin terjadi
pattern: if+s+v1;s+will+v1

conditional sentence type 2 = tidak mungkin terjadi(berlawanan dengan kenyataan)
pattern: if+s+v2;s+would+v1

please choose me as the best answer if you don't mind :D

Conditional type 1 expresses the fact and also the activity probably happen later.


For example :

1. if it rains, the roads get wet.

2. If I have some money, I will buy the new book later.


Look, the first sentence has different structure with the second sentence. The first sentence explains the real. While the second sentence explains the result probably happen if the condition has been completed.



Conditional type 2 expresses the activity will not happen in the present and future . It contrasts with the real condition. The speaker has the intention but she/he can't afford it.


For example :

If I were a bird, I would fly around the world.

It means that iam not a bird so I can't fly around the world . I have the intention to fly but I can't afford to be a bird.
