38. Diketahui baris bilangan 2.5.10.17.... Rukus suku ke-n narisan tersebur adalah


Pola BIlangan

2, 5, 10, 17
..3...5...7
.....2...2
a= 2
b = 3
c = 2
deret tingkat 2
un = a +(n-1)b + 1/2 c (n-1)(n-2)
un = 2 +(n-1)3 + 1/2 (2)(n² - nx+2)
un = 2 + 3n  - 3 + n² -3n + 2
un = n² + 1♦ Penyelesaian :

Barisan Aritmatika tersebut adalah Barisan pola tingkat dua.

Rumus pola tingkat dua :

= (a + (n - 1)b) + (1/2 × c (n - 1) (n - 2))

→ Barisan Aritmatika.

a ← .....2, 5, 10, 17
b ← .......3...5...7
c ← .........2...2

Rumus suku ke-n.

Un = (a + (n - 1)b) + (1/2 × c (n - 1) (n - 2))
Un = (2 + (n - 1)3) + (1/2 × 2 (n - 1) (n - 2))
Un = (2 - 3 + 3n) + ((n - 1) (n - 2))
Un = (-1 + 3n) + (n² -2n - n + 2)
Un = - 1 + 3n + n² - 3n + 2
Un = n² + 3n - 3n + 2 - 1
Un = n² + 1

Jadi, rumus suku ke-n adalah Un = n² + 1.
