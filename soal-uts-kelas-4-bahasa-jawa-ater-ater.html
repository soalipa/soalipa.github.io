PenjelasanKajaba tegese kecuali.

{Ater-ater}

→ yaiku wuwuhan kang manggon ing sangarepe tembung.

1. Ater-ater anuswara (n, m, ny, ng). Ater-ater iki gunane ngubah tembung lingga dadi tembung kriya.

» Tuladhane :

ny + sapu → nyapun + tulis → nulisng + langi → nglangim + pacul → macul

2. Ater-ater tripurusa (dak, ko/kok, di). Ater-ater tripurusa gunane kanggo tembung sesulih aku (dak) kowe (ko) lan dheweke (di).

» Tuladhane :

dak+tulis → daktuliskok+pangan → kokpangandi+pancing → dipancing

3. Ater-ater liyane kayata sa, pa, pi, ka, kuma, kami, a, ma, lan sapanunggalane.

» Tuladhane :

satus → sa + atuspanulis → pa + nulispituduh → pi + tuduhkapidek → ka + pidekkumawani → kuma + wanikamitenggeng → kami + tenggengmanepi → ma + nepi

29. 1. Kelas 4 tabuh gamelan ing wayah awan tembung lingga tabuh iku mathuke oleh wuwuhan wujud.. A ater - ater m-B ater - ater ng-C ater - ater nD ater - ater ny-2. Wangsulan

1. Kelas 4 tabuh gamelan ing wayah awan tembung lingga tabuh iku mathuke oleh wuwuhan wujud..

➡️ C. ater-ater n.

2. Ibu nyawisake dhaharan kanggo bapak. Nyawisake yen diudal manut tembung linggane..

➡️ Ater-ater ny- + cawis + panambangan -ake.

Penjelasan

>> Ater-ater

→ yaiku wuwuhan kang manggon ing sangarepe tembung.

1. Ater-ater anuswara (n, m, ny, ng). Ater-ater iki gunane ngubah tembung lingga dadi tembung kriya.

» Tuladhane :

ny + sapu → nyapun + tulis → nulisn + tabuh → nabuhng + langi → nglangim + pacul → macul

2. Ater-ater tripurusa (dak, ko/kok, di). Ater-ater tripurusa gunane kanggo tembung sesulih aku (dak) kowe (ko) lan dheweke (di).

» Tuladhane :

dak+tulis → daktuliskok+pangan → kokpangandi+pancing → dipancing

3. Ater-ater liyane kayata sa, pa, pi, ka, kuma, kami, a, ma, lan sapanunggalane.

» Tuladhane :

satus → sa + atuspanulis → pa + nulispituduh → pi + tuduhkapidek → ka + pidekkumawani → kuma + wanikamitenggeng → kami + tenggengmanepi → ma + nepi

>> Panambang (Akhiran)

→ Panambang yaiku wuwuhan sing manggon ana ing mburine tembung. Panambang iku sabenere ana akeh maceme, ning ing kene mung dirembug sing kerep digunakake kaya dene -ku, -mu, -e, -an, -i, -a, -na, -ana, -en, lan -ake.

» Tuladha :
