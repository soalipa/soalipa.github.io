Banyak sisi kubus sebelum diiris

= 6 + 8 ( karena yang diiris ada 8 sudut )

= 14

14 + 6

= 20

Jadi nilai m + n adalah 20

11. Tolong bantu kak beserta dgn langkah langkah MATERI :VOLUME BANGUN RUANG KLS:6 SD


Soal 1

Volume gabungan

= Volume kerucut + Volume tabung

= ( Luas alas × t ) + ( ⅓ × L. alas × t )

= ( ( π × r² ) × t ) + ( ⅓ × ( π × r² ) × t )

= ( ( 22/7 × 7² ) × 12 ) + ( ⅓ × ( 22/7 × 7² ) × 6 )

= ( 154 × 12 ) + ( ⅓ × 924 )

= 1.848 cm³ + 308 cm³

= 2.156 cm³

Soal 2

Volume balok

= Luas alas × t

= ( P × L ) × t

= ( 10 × 5 ) × 4

= 50 × 4

= 200 cm³
