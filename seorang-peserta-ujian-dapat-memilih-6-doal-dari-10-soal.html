17. contoh doal konstanta pegas


contoh soal :
1. Lima pegas masing-masing mempunyai konstanta 200 N/m disusun secara seri. Konstanta pegas pengganti dari susunan pegas tersebut adalah…

Pembahasan
Diketahui :
Konstanta masing-masing pegas (k1 = k2 = k3 = k4 = k5) = 200 N/m
Ditanya : konstanta pegas pengganti (k)
Jawab :

Konstanta pegas pengganti :
1/k = 1/k1 + 1/k2 + 1/k3 + 1/k4 + 1/k5
1/k = 1/200 + 1/200 + 1/200 + 1/200 + 1/200
1/k = 5/200
k = 200/5
k = 40 Newton/meter

KELAS : IV

MAPEL : MATEMATIKA

BAB : Bilangan Bulat

KATA KUNCI : Operasi Hitung Bilangan Bulat Dalam Bentuk Soal Cerita

KODE KATEGORISASI : 4.2.5

---------------------------------------------------------------------

Pembahasan Soal :

Dalam Sebuah ujian pilihan ganda, terdapat aturan jika jawaban benar diberi skor 4, salah diberi -1 dan tidak dijawab diberi skor 0. Dari 45 doal, Deisy menjawab 35 soal benar, 2 soal

45 doal, Deisy menjawab 35 soal benar, 2 soaltidak dijawab dan sisanya salah. Berapakah total skor Deisy?

Pembahasan Jawaban :

Diketahui :

Nilai Soal Benar = 35 x 4 = 140Nilai Soal Tidak Dijawab = 2 x 0 = 0Nilai Soal Salah = 8 x -1 = -8

Ditanya :

Total Skor.....?

Dijawab :

S = Nilai Soal Benar + Nilai Soal Tidak Dijawab + Nilai Soal Salah

S = 140 + 0 + (-8)

S = 140 + (-8)

S = 132

Jadi, Total skor Deisy adalah 132
